# Scoring Rubric

Blends automated flag capture with report-based evidence grading.

| Component | Weight (proposed) | Scored by |
|-----------|-------------------|-----------|
| Dynamic flag capture | 40% | Automated per-team flags |
| Evidence objectives | 25% | Report vs. required evidence checklist |
| Reproduction quality | 15% | Complete, ordered, repeatable steps |
| Detection awareness | 10% | Telemetry/detection opportunity identified |
| Remediation & risk | 10% | CWE mapping, defensible severity, actionable fix, residual risk |

Per-finding 0–4: 0 not attempted; 1 surface only; 2 validated, weak evidence; 3 validated +
solid evidence/repro; 4 validated, reproduced, mapped (CWE/CVSS/ATT&CK), with detection + fix.

Deductions: out-of-scope activity; overstated impact; flags without evidence; unauthorized
destructive actions. Submission: one PDF report per team + flags. Instructor solution guides
(private) calibrate grading.
