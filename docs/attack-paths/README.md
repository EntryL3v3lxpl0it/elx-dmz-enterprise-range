# Attack Paths (index)

Design-level only in current phases. Each future path will carry: student objective,
instructor-only solution (private), evidence checklist, detection opportunities, remediation,
and standards mapping (CWE / CVSS / MITRE ATT&CK). No exploit code or solution walkthroughs
are published here.

Planned chained paths:
1. External web exposure -> internal data access
2. Helpdesk weakness -> credential exposure
3. DMZ foothold -> AD enumeration
4. Cloud misconfiguration -> secret exposure
5. Internal service weakness -> privilege escalation
6. Weak RBAC -> unauthorized business-data access
